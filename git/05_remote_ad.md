## 원격저장소 활용

### clone

원격 저장소를 복제하여 로컬에서 활용할 수 있도록 한다.

```bash
$ git clone __url__
```

- 원격 저장소 이름의 폴더가 생성되고, 해당 폴더로 이동하면 git을 활용할 수 있다.

## push & pull

```bash
$ git push origin master
```

```bash
$ git pull origin master
```
