# Branch

## 명령어

1. 브랜치 생성

   ```bash
   $ git branch __브랜치이름__
   ```

2. 브랜치 이동

   ```bash
   $ git checkout __브랜치이름__
   ```

3. 브랜치 생성 및 이동

   ```bash
   $ git checkout -b __브랜치이름__
   ```

4. 브랜치 목록

   ```bash
   $ git branch
   ```

5. 브랜치 병합

   ```bash
   (master) $ git merge __브랜치이름__
   ```

   * master 브랜치에 브랜치이름을 병합시킨다.

6. 브랜치 삭제

   ```bash
   $ git branch -d __브랜치이름__
   ```
